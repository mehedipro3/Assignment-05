Important FAQ

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

how do you select an element from the DOM ?

The click() method simulates a mouse-click on an element. This method can be used to execute a click on an element as if the user manually clicked on it.

what is event delegation in the context of the dom and why is it useful ?
